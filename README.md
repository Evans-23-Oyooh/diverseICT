# Diverse ICT College Website

A modern, responsive website for Diverse ICT College featuring comprehensive information about academic programs, admissions, and campus life.

## Features

- **Responsive Design**: Mobile-first approach with professional sidebar navigation
- **Modern UI**: Clean, professional design with gradient backgrounds and animations
- **Interactive Elements**: Image sliders, dropdown menus, and smooth transitions
- **Comprehensive Content**: Complete information about programs, admissions, and college services
- **Professional Navigation**: Icon-enhanced menu system with mobile-optimized sidebar

## Pages Included

- Homepage with hero slider
- About Us with mission and vision
- Academic Programs (KASNEB & KNEC certified)
- Admissions information and requirements
- Contact page with forms
- Leadership team profiles
- Library services
- Academic calendar
- Student life and facilities

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Font Awesome Icons
- Unsplash Images

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Navigate through the website using the menu

## Mobile Features
